6. SETUP - HEADLESS INJECT
--------------------------

Spust v novem terminalu (source setup.bash neni potreba, jde o plain ros2 CLI):

```bash
cd /home/tj/_Data/_Projekty/TJlabs/scout_ws
source /opt/ros/jazzy/setup.bash && source install/setup.bash

# pad_0
ros2 topic pub --once /swarm/pad_assignment std_msgs/String \
  '{data: "{\"drone_id\":\"drone_0\",\"pad_id\":\"pad_0\",\"x\":0.0,\"y\":-5.0,\"z\":-0.5}"}'

# pole - 4 vertexy
ros2 topic pub --once /field/boundary_point std_msgs/String \
  '{data: "{\"index\":0,\"ned\":{\"x\":5.0,\"y\":0.0,\"z\":-5.0},\"type\":\"vertex\"}"}'
ros2 topic pub --once /field/boundary_point std_msgs/String \
  '{data: "{\"index\":1,\"ned\":{\"x\":5.0,\"y\":20.0,\"z\":-5.0},\"type\":\"vertex\"}"}'
ros2 topic pub --once /field/boundary_point std_msgs/String \
  '{data: "{\"index\":2,\"ned\":{\"x\":35.0,\"y\":20.0,\"z\":-5.0},\"type\":\"vertex\"}"}'
ros2 topic pub --once /field/boundary_point std_msgs/String \
  '{data: "{\"index\":3,\"ned\":{\"x\":35.0,\"y\":0.0,\"z\":-5.0},\"type\":\"vertex\"}"}'

# uzavri polygon -> zapise field_boundary.json + field_grid.json
ros2 topic pub --once /field/boundary_close std_msgs/String \
  '{data: "{\"source\":\"manual_topic\"}"}'

# over ze soubory existuji
cat perimeters/home_positions.json | python3 -m json.tool
cat perimeters/field_boundary.json | python3 -m json.tool
```

TROUBLESHOOTING
---------------

Dron se po mission_confirm nehybe:

```bash
ros2 topic echo --once /drone_0/avoidance/status | grep -E 'phase|state|planner'
ros2 topic hz /drone_0/depth/image_raw
```

  planner_state=DEGRADED -> depth netece nebo mapper nema pose; over hz depth
  phase=BLOCKED -> viz blocked_reason v status

Depth netece -> v Isaac Script Editoru spust simulation_cam.py (jen jednou!)

PX4 "Waiting for simulator...":
  -> ROS2 backend nezapinaj pred Isaac Play

vic nez 1 publisher na /fmu/in/trajectory_setpoint:
  -> zustavaji bezet dva backendy paralelne; ukonci Phase 1+2 pred spustenim Phase 3
