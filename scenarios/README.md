# Scenarios

This folder is consumed by `scout_launcher.py`.

## Current Production Scenario

- `full_e2e_mission.yaml` - the current Phase 1-5 / Phase 15 Gazebo swarm E2E
  milestone scenario.

Use this scenario for the final demonstrated workflow.

## Legacy Scenarios

The `legacy/` folder contains older single-purpose, partial-phase, and debug
scenarios. They are intentionally not presented as the current project path.
Keep them for diagnostics, regression experiments, or historical reference.

Do not use a legacy scenario as evidence for the final milestone unless it is
explicitly revalidated and promoted back to this folder.
